# DigiTools - Your All-in-One Digital Solution

A modern web application built with *React* and *Tailwind CSS* that provides essential digital tools for everyday tasks.

## Key Features
* Fast and Responsive UI.
* Built with modern web technologies.
* Optimized for performance using Vite.

## Tech Stack
* *Frontend:* React.js
* *Styling:* Tailwind CSS
* *Build Tool:* Vite
* *Deployment:* Vercel

*Live Site:* https://digitools-blush.vercel.app/
